# Raga - Jurnal & Latihan Gym Premium

Aplikasi mobile-first pelacak latihan gym premium, volume, PR, progres fisik, dan progressive overload.

## Fitur Utama

- **Pencatatan Latihan & Target Rencana Latihan**: Catat gerakan latihan, set, repetisi, dan RIR (*Reps in Reserve*).
- **Progresive Overload Tracking**: Sistem secara otomatis mengambil data historis set beban latihan dari minggu sebelumnya sebagai default.
- **Visualisasi Stimulus Otot**: Grafik radar pembagian volume set latihan per grup otot secara mingguan.
- **Photo Gallery Progres**: Unggah foto progres fisik Anda dengan catatan detail untuk melacak perkembangan fisik visual Anda.
- **Log Kalori & Nutrisi Harian**: Pelacak diet harian lengkap dengan makronutrisi (Protein, Karbo, Lemak) dan target kalori (TDEE) yang dapat disesuaikan per hari.
- **Circular Progress Loading**: Desain tampilan pemuatan awal (onboarding) yang dinamis dan premium.

## Mulai Pengembangan Lokal

Jalankan perintah berikut untuk memulai server pengembangan:

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) pada browser Anda untuk melihat hasilnya.
